Framer Motion 3D is an animation library that ties together

It looks like this:

```jsx
<motion.boxGeometry animate={{ x: 0 }} />
```

It does all this:

-   Springs
-   Keyframes
-   Gestures (tap/hover)
-   Exit animations
-   Orchestrate animations across components

...and a whole lot more.
